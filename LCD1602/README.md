> # LCD1602 驱动库说明文档
>
> ## 概述
>
> 本库提供了基于 STM32 HAL 库的 LCD1602 字符液晶显示屏驱动功能，支持基本的初始化、指令写入、数据显示及字符串显示等操作。
>
> ## 硬件连接说明
>
> - **控制引脚**：RS、RW、E 连接至 GPIOA
> - **数据引脚**：D0-D7 连接至 GPIOB
> - 引脚定义需在工程中通过宏定义（如`RS_Pin`、`RW_Pin`等）指定具体引脚编号
>
> ## 功能函数说明
>
> ### 1. 方向控制函数
>
> 
>
> ```c
> void DataDir(char dir)
> ```
>
> - 功能：设置 D0-D7 数据引脚的方向
> - 参数：
>   - `dir`：'I' 表示输入模式，'O' 表示输出模式
>
> ### 2. 数据读写函数
>
> 
>
> ```c
> uint8_t ReadData()
> ```
>
> - 功能：从 D0-D7 引脚读取 8 位数据
> - 返回值：读取到的 8 位数据
>
> 
>
> ```c
> void WriteData(uint8_t dat)
> ```
>
> - 功能：向 D0-D7 引脚写入 8 位数据
> - 参数：`dat`：待写入的 8 位数据
>
> ### 3. 忙等待函数
>
> ```c
> void LCD_Busy_Wait()
> ```
>
> - 功能：等待 LCD 控制器空闲（检测忙标志位）
> - 说明：通过读取状态寄存器的最高位判断 LCD 是否繁忙
>
> ### 4. 指令与数据写入函数
>
> 
>
> ```c
> void LCD_Write_Cmd(uint8_t cmd)
> ```
>
> - 功能：向 LCD 写入指令
> - 参数：`cmd`：待写入的指令代码
>
> 
>
> ```c
> void LCD_Write_Data(uint8_t dat)
> ```
>
> - 功能：向 LCD 写入显示数据
> - 参数：`dat`：待显示的字符 ASCII 码
>
> ### 5. 初始化函数
>
> 
>
> ```c
> void LCD_Init()
> ```
>
> - 功能：初始化 LCD1602 显示屏
> - 初始化流程：
>   - 设置 8 位数据接口、2 行显示模式
>   - 清屏操作
>   - 设置光标移动方向（右移，屏幕不滚动）
>   - 开启显示，关闭光标
>
> ### 6. 字符串显示函数
>
> 
>
> ```c
> void LCD_ShowString(uint8_t x, uint8_t y, char *str)
> ```
>
> - 功能：在指定位置显示字符串
> - 参数：
>   - `x`：行号（0 表示第一行，1 表示第二行）
>   - `y`：列号（0-15，表示一行中的位置）
>   - `str`：待显示的字符串（以 '\0' 结尾）
> - 说明：最多显示 16 个字符，超出部分将被截断
>
> ## 使用示例
>
> 
>
> ```c
> // 初始化LCD
> LCD_Init();
> 
> // 在第0行第0列显示"Hello"
> LCD_ShowString(0, 0, "Hello");
> 
> // 在第1行第3列显示"World"
> LCD_ShowString(1, 3, "World");
> ```
>
> ## 注意事项
>
> 1. 函数中使用了`HAL_Delay()`进行延时，需确保系统滴答定时器已正确初始化
> 2. 引脚宏定义（如`RS_Pin`、`D0_Pin`等）需在工程中根据实际硬件连接进行配置
> 3. 字符串显示时需注意屏幕尺寸限制（16x2），避免越界显示
> 4. 所有操作均包含忙等待机制，确保指令执行完成后再进行后续操作
